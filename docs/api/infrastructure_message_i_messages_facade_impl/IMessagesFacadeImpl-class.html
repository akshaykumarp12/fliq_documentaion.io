IMessagesFacadeImpl

IMessagesFacadeImpl class

Implementation of the IMessagesFacade interface that handles all message-related operations. This class serves as the main entry point for message operations in the application, coordinating between local storage, remote API calls, and message events.

The implementation follows the Repository pattern and handles:

  • Message persistence
  • Message retrieval
  • Message status updates
  • Vanish mode operations
  • Sticker and gift catalog management
