SocialAccountDetails toAccountDetails<T>(
  1. T param
)

Converts the raw response from a social sign-in provider into a SocialAccountDetails object.

param - The raw response from the provider. Supported types are:

  • GoogleSignInAccount for Google sign-in.
  • AuthorizationCredentialAppleID for Apple sign-in.
  • Map<String, dynamic> for Instagram sign-in.

Returns a SocialAccountDetails object containing standardized data.

Throws a CustomException if the type of param is not supported.

Implementation

static SocialAccountDetails toAccountDetails<T>(T param) {
  if (param is GoogleSignInAccount) {
    // Maps Google sign-in details to SocialAccountDetails.
    return SocialAccountDetails(
      email: param.email,
      name: param.displayName,
      picture: param.photoUrl,
      socialId: param.id,
    );
  } else if (param is AuthorizationCredentialAppleID) {
    // Maps Apple sign-in details to SocialAccountDetails.
    return SocialAccountDetails(
      email: param.email,
      name: param.givenName,
      picture: null, // Apple sign-in does not provide a profile picture.
      socialId: param.userIdentifier,
    );
  } else {
    // Throws an exception for unsupported provider types.
    throw CustomException(
      errMsg: 'TypeError ${T.toString()} Provider Not Found',
    );
  }
}
