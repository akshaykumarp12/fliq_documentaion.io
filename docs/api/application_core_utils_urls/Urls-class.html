Urls

Urls class

Centralized collection of API endpoint URLs for the application.

This class provides static constants and variables representing base URLs, versioning, and specific REST API endpoint paths used throughout the app.

The base URL is dynamically obtained from the app environment configuration, allowing flexibility between development, staging, and production environments.

Example usage of building full endpoint URLs:

final loginUrl = '${Urls.apiVersionUrl}${Urls.auth}/login';
