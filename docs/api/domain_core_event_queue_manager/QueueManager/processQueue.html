Processes the given queue of entries sequentially using the provided handler.

Parameters:

  • queue: The queue containing items to process, each item is a MapEntry of generic type T and a String key.
  • handler: An asynchronous function to handle processing of each queue entry.
  • isProcessingGetter: A function to check if processing is already ongoing. If true, the method returns immediately.
  • setProcessingFlag: A function to set the processing flag status before and after processing.
  • onError: A callback invoked with the error object if processing an entry fails.

Throws no exceptions and logs errors instead, allowing processing to continue with next items on failure.

Future<void> processQueue<T>({
  required Queue<MapEntry<T, String>> queue,
  required Future<void> Function(MapEntry<T, String>) handler,
  required bool Function() isProcessingGetter,
  required void Function(bool) setProcessingFlag,
  required void Function(dynamic) onError,
}) async {
  if (isProcessingGetter()) return;
  setProcessingFlag(true);

  while (queue.isNotEmpty) {
    final entry = queue.removeFirst();
    Logger.logInfo(
        'Processing queue: ${entry.key.runtimeType} ${queue.length} items remaining');
    try {
      await handler(entry);
    } catch (e) {
      onError(e);
      Logger.logError('Processing queue failed: $e');
    }
  }

  setProcessingFlag(false);
}
