Called whenever the object needs to paint. The given Canvas has its coordinate space configured such that the origin is at the top left of the box. The area of the box is the size of the size argument.

Paint operations should remain inside the given area. Graphical operations outside the bounds may be silently ignored, clipped, or not clipped. It may sometimes be difficult to guarantee that a certain operation is inside the bounds (e.g., drawing a rectangle whose size is determined by user inputs). In that case, consider calling Canvas.clipRect at the beginning of paint so everything that follows will be guaranteed to only draw within the clipped area.

Implementations should be wary of correctly pairing any calls to Canvas.save/Canvas.saveLayer and Canvas.restore, otherwise all subsequent painting on this canvas may be affected, with potentially hilarious but confusing results.

To paint text on a Canvas, use a TextPainter.

To paint an image on a Canvas:

  1. Obtain an ImageStream, for example by calling ImageProvider.resolve on an AssetImage or NetworkImage object.

  2. Whenever the ImageStream's underlying ImageInfo object changes (see ImageStream.addListener), create a new instance of your custom paint delegate, giving it the new ImageInfo object.

  3. In your delegate's paint method, call the Canvas.drawImage, Canvas.drawImageRect, or Canvas.drawImageNine methods to paint the ImageInfo.image object, applying the ImageInfo.scale value to obtain the correct rendering size.

@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..color = Colors.pink
    ..strokeWidth = 3;
  final shadowPaint = Paint()
    ..color = Colors.blueGrey.withOpacity(0.2) // Shadow color
    ..maskFilter = MaskFilter.blur(BlurStyle.outer, 4);

  final whiteDotPaint = Paint()
    ..color = Colors.white
    ..style = PaintingStyle.fill;

  final pinkDotPaint = Paint()
    ..color = Colors.pink
    ..style = PaintingStyle.fill;

  final textStyle = context.textStyle;

  final subTextStyle = context.textStyle.s12;

  const double lineHeight = 90;
  double lineStartX = size.width * 0.5;
  const double dotRadius = 10;
  const double pinkDotRadius = 5; // Smaller pink dot

  if (progress.deliveryStatus?.name != 'delivered' &&
      progress.deliveryStatus?.name != 'returned') {
    canvas.drawLine(
      Offset(lineStartX, 10),
      Offset(lineStartX, lineHeight),
      paint,
    );
  }

  if (progress.deliveryStatus?.name == 'returned') {
    final datePainter = TextPainter(
      textAlign: TextAlign.center,
      text: TextSpan(
        text: 'Product Returned',
        style: context.textStyle.red,
      ),
      textDirection: TextDirection.ltr,
    );
    datePainter.layout();

    final canvasWidth = size.width;
    final canvasHeight = size.height;

    final xPosition = (canvasWidth - datePainter.width) / 2;
    final yPosition = (canvasHeight - datePainter.height) / 4;

    // Paint the text at the calculated center
    datePainter.paint(canvas, Offset(xPosition, yPosition));
    return;
  }

  canvas.drawCircle(Offset(lineStartX, 0), dotRadius, whiteDotPaint);
  canvas.drawCircle(Offset(lineStartX, 0), dotRadius, shadowPaint);

  canvas.drawCircle(Offset(lineStartX, 0), pinkDotRadius, pinkDotPaint);

  // Draw text with adjusted horizontal spacing
  final datePainter = TextPainter(
    textAlign: TextAlign.end,
    text: TextSpan(
      text: progress.createdAt.ddMMMyyy.trim(),
      style: context.textStyle,
    ),
    textDirection: TextDirection.ltr,
  );
  datePainter.layout();
  datePainter.paint(
      canvas, Offset(lineStartX - 112, -10)); // Increased x offset

  final timePainter = TextPainter(
    textAlign: TextAlign.end,
    textWidthBasis: TextWidthBasis.longestLine,
    text: TextSpan(
      text: progress.createdAt.time.trim(),
      style: subTextStyle,
    ),
    textDirection: TextDirection.ltr,
  );
  timePainter.layout();
  timePainter.paint(canvas, Offset(lineStartX - 95, 13));

  final titlePainter = TextPainter(
    text: TextSpan(
      text: '${progress.deliveryStatus?.name}'.capitalize,
      style: textStyle,
    ),
    textDirection: TextDirection.ltr,
  );
  titlePainter.layout();
  titlePainter.paint(canvas, Offset(lineStartX + 30, -10));

  // final subtitlePainter = TextPainter(
  //   text: TextSpan(
  //     text: '${progress.statusLocation}'.capitalize,
  //     style: subTextStyle,
  //   ),
  //   textDirection: TextDirection.ltr,
  // );
  // subtitlePainter.layout();
  // subtitlePainter.paint(canvas, Offset(lineStartX + 30, 13));
}
