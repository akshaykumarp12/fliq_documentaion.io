Clears local persistent storage and token, optionally navigates to login.

Used internally in DioClient to handle unauthorized (401) errors.

Implementation

Future<void> clearLocalDb(bool navigate) async {
  await sl<SharedPreferences>().clear();

  await sl<LocalDb>().clearDB();
  sl<FirebaseMessaging>().deleteToken();

  if (navigate) {
    AppRoute.pushNamed(routeName: Login.routeName);
  }
}
