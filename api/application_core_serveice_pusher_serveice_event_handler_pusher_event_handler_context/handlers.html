Map<String, PusherEventHandler> handlers
getter/setter pair

A mapping of Pusher event names to their corresponding event handlers.

Each event name (as string) corresponds to an instance of a class implementing PusherEventHandler. The handler processes the specific event data received from Pusher.

Using dependency injection (sl) to resolve handlers ensures clean initialization and decoupling.

Implementation
